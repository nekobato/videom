videom
======
collect videos, play on iPad


see dir

* tools - downloader

works with 

* Ruby 1.8.7
* Mac OSX or Linux
* Sinatra 1.3+
* MongoDB 2.0+


config
------

    % cp sample.config.yaml config.yml

edit config.yml.


Run
---

    % ruby development.ru

open [http://localhost:8125](http://localhost:8125)


Deploy
------
use Passenger with "config.ru"
